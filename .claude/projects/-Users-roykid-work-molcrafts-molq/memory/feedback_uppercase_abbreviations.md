All abbreviations must use full uppercase, e.g. PBS not Pbs, LSF not Lsf.

**Why:** User preference for consistent, standard abbreviation casing.

**How to apply:** When naming classes, variables, methods, or writing comments/docstrings that include abbreviations like PBS, LSF, SLURM, HPC — always use full uppercase.
